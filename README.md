# Dark Web Threat Intelligence Crawler

## Description
This project is a Python-based crawler designed to collect and analyze .onion (dark web) links using the Tor network.

## Features
- Routes traffic through Tor using SOCKS5 proxy
- Scrapes onion links using BeautifulSoup
- Filters and validates links
- Removes duplicates
- Stores results in CSV format

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Tor (SOCKS5 Proxy)
- Pandas

## Use Case
This project can be used for basic threat intelligence and identifying hidden or suspicious services on the dark web.
